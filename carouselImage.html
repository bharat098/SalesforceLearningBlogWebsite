Carousel Image

Published on 05/09/20235 min read4 comments

In this component we are showing all the images in a carousal using noraml bootstrap in salesforce org,community(Digital Experience). Below is the step by step process how to Start

Upload The Static Resources

For the below 3 Files Download the static resources file Here.

1. From Setup, enter Static Resources in the Quick Find box, then select Static Resources, and then click New.

2. Enter bootstrapminjs for the Name.Click Choose File, and then choose the bootstrapminjs.js file.

3. If you see the Cache Control menu, choose Public. This item isnâ€™t visible in all organizations.

4. Click Save.You now have a static resource version of bootstrapminjs that you can use in your carousal.

Follow the Steps 1-4 for other 2 files as well.

For more Guidance on how to upload staticResources. Please follow this Trailhead.

Create a Apex class named InstituteFileController

InstituteFileController.cls
					    
public with sharing class InstituteFileController{

	public List imageUrls { get; set; }
	public List imageUrlList { get; set; }
	public String image { get; set; }
	
	public InstituteFileController() {
		imageUrlList = new List();
		imageUrls = populateImageUrls();
		for(integer i = 0;i populateImageUrls() {
		List imageUrls = new List();
		Institute__c instituteRecord = [SELECT Id FROM Institute__c  LIMIT 1];
		// Replace 'Your_Object_Id_Here' with the actual object Id you want to associate Attachments with
		List attachments = [SELECT Id, Name,ContentType FROM Attachment WHERE ParentId =: instituteRecord.Id ];
		for (Attachment attachment : attachments) {
			String imageUrl ='/servlet/servlet.FileDownload?file=' + attachment.Id;
			imageUrls.add(imageUrl);
		}
		return imageUrls;
	}
}

Create the VisualForce Page Named ImageSlider

ImageSlider.page
						
<apex:page showHeader="false" controller="InstituteFileController">

<html lang="en">
<head>
	<title>Bootstrap Example</title>
	<meta charset="utf-8"/>
	<meta name="viewport" content="width=device-width, initial-scale=1"/>
		<apex:stylesheet value="{!URLFOR($Resource.bootstrapmincss)}"/>
			<apex:includeScript value="{!URLFOR($Resource.jquery3_6)}"/>
			<apex:includeScript value="{!URLFOR($Resource.bootstrapminjs)}"/>
			<apex:stylesheet value="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.3.0/css/all.min.css"/>
	</head>
<body>

<div class="container" style="width:100%">
	
	<div id="myCarousel" class="carousel slide" data-ride="carousel">
	<!-- Indicators -->
	<ol class="carousel-indicators">
		<li data-target="#myCarousel" data-slide-to="0" class="active"></li>
		<apex:variable value="{!0}" var="index"/>
		<apex:repeat value="{!imageUrlList}" var="image">
			<li data-target="#myCarousel" data-slide-to="{!index + 1}" />
			<apex:variable value="{!index + 1}" var="index"/>
		</apex:repeat>
	</ol>

	<div class="carousel-inner">
		<div class="item active">
		<img src="{!image}" alt="Image" style="width:100% !important;height:390px;"/>
	</div>
		<apex:repeat value="{!imageUrlList}" var="image" >
			<div class="item">
				<img src="{!image}" alt="Image" style="width:100% !important;height:390px;"/>
			</div>
		</apex:repeat>
	</div>

	
	<!-- Left control -->
	<a class="left carousel-control" href="#myCarousel" data-slide="prev">
	<span class="fa fa-chevron-left" style="margin-top: 70%;"></span>
	</a>

	<!-- Right control -->
	<a class="right carousel-control" href="#myCarousel" data-slide="next">
	<span class="fa fa-chevron-right" style="margin-top: 70%;"></span>
	</a>
	
	</div>
</div>

</body>

</html>

</apex:page>
