What I do

I have more than 3 years' experience building Salesforce software for clients all over the world. Below is a quick overview of my main technical skill sets and technologies I use.

Salesforce

Salesforce is a cloud-based customer relationship management (CRM) platform that helps businesses manage and streamline their sales, customer service, marketing, and more.

LWC

LWC is a modern framework for building web components on the Salesforce platform, making it easier to create dynamic and responsive user interfaces.

Lightning-Aura

Lightning Aura is a component-based framework in Salesforce used for developing custom user interfaces and applications that are highly customizable and interactive.

Apex

Apex is a programming language specific to the Salesforce platform. It's used for building custom business logic, automation, and extensions within Salesforce applications.

JavaScript

JavaScript is a scripting language used for adding interactivity and dynamic behavior to websites. It's essential for creating interactive web applications.

Bootstrap

Bootstrap is a popular front-end framework that provides pre-designed, responsive, and customizable components and styles to streamline web development.

Java

Java is a widely-used, object-oriented programming language known for its portability and versatility. It's often used in web development, server-side applications, and more.

HTML & CSS

HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) are fundamental web technologies. HTML structures web content, while CSS is used for styling and layout, ensuring your website looks and behaves as desired.
